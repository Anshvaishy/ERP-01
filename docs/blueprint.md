# **App Name**: EduConnect

## Core Features:

- Landing Page: Display key information about the university including hero section, courses, stats, testimonials, and a footer with contact details.
- Student Login: Authenticate students using Firebase and redirect them to their dashboard.
- Student Registration: Allow new students to register by providing their personal details and course selection, storing the data in Firebase.
- Student Dashboard: Show personalized student information such as welcome message, profile details, attendance, marks, and announcements.
- Feedback Submission: Collect student feedback, store it in Firebase, and display a confirmation message.
- Contact Information: Present contact details and a contact form, including a university address and map placeholder.
- Theme Customization: Switch between dark and light themes for enhanced user experience.

## Style Guidelines:

- Primary color: A deep, stately blue (#1E3A8A), suggestive of trust, stability, and intelligence which are all virtues a student might seek. This contrasts well with a light or dark background.
- Background color: A very light blue (#F0F9FF) creates a calm and professional feel.
- Accent color: A vibrant, slightly lighter shade of blue (#3B82F6) for interactive elements and calls to action to draw the user's attention.
- Body and headline font: 'PT Sans', a modern sans-serif font that will contribute warmth and personality to the site.
- Use simple, outlined icons to represent different sections and actions.
- Employ a clean, grid-based layout with ample white space for easy navigation and readability.
- Incorporate subtle animations on hover and page transitions to enhance user engagement.
